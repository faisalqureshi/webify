# Installation instructions

## Without Docker

Install Anaconda python distribution.

conda create -n webify python=2.7
activate webify
conda install -c conda-forge pypandoc=1.4
conda install pyyaml=3.13
pip install pystache==0.5.4

You can use webify within the webify environment.

## Using docker

The preferred way of using webify is through docker.
This allows us to use webify in any environment that supports docker.
Check out the instructions available in the docker/ folder.
