# Installation instructions

## Webify >= 3.0 (Uses Python 3.  Dec 9, 2019)

### Installation instructions

Webify requires pandoc (version > 1.19) and LaTeX (for pdf generation).

#### OSX

Install Homebrew first.

$ brew install python3
$ pip install -r requirements.txt

#### Linux

$ apt-get install python3
$ pip install -r requirements.txt

#### Windows (Anaconda)

I recommend using the Ubuntu shell available in Windows to 
install Webify.
